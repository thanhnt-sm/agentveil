🚀 Quick Start

Agent Veil is a transparent security proxy that sits between your AI tools and LLM APIs. It automatically protects PII, blocks prompt injection attacks, and audits skills for security risks.

# All AI traffic is already routed through the proxy:
# Antigravity / Claude Code → http://localhost:8080 → Anthropic API
# Cursor / Aider            → http://localhost:8080/v1 → OpenAI API
# Gemini SDK                → http://localhost:8080/gemini → Google API

🛡️ Security Layers

  • PII Shield — Detects and anonymizes CCCD, CMND, Phone, Email, SSN, Credit Card, API keys before sending to LLMs
  • Prompt Injection Guard — Blocks 11+ attack patterns including DAN mode, instruction override, jailbreak attempts
  • Skill Auditor — Static analysis detects reverse shells, data exfiltration, dangerous commands in SKILL.md files
  • AES-256-GCM Vault — Encrypted token storage in Redis with per-session isolation and 30-minute TTL
  • Role-Based Masking — admin sees full data, viewer sees 70% masked, operator sees partial
  • Rate Limiting — Per-IP sliding window (60 req/min, burst 20)
  • Runtime Guardrails — Token limits, harmful content blocking, session duration limits

🔧 CLI Commands

# Audit a skill file for security risks
agentveil audit path/to/SKILL.md

# Scan text for PII
agentveil scan "CCCD: 012345678901, email: test@example.com"

# Check compliance
agentveil compliance check --framework vietnam
agentveil compliance check --framework all

# Service management
./setup.sh --status      # Check all components
./setup.sh --restart     # Rebuild + restart
./setup.sh --logs        # Tail proxy logs
./setup.sh --stop        # Stop proxy
./setup.sh --start       # Start proxy

🤖 Automation (Watchdog)

The watchdog runs every 15 minutes automatically via launchd:

  • Health check — Verifies proxy and Redis are running
  • Auto-restart — If proxy is down, automatically reloads the launchd service
  • Skill scanning — Audits all SKILL.md files and reports risks
  • Log rotation — Keeps logs under 10MB
  • Reports — JSON reports saved to ~/.agentveil/reports/

🌐 Multi-Provider Routing

Agent Veil routes API requests to the best available provider with automatic failover:

# Router config: ~/.agentveil/router.yaml
providers:
  - name: anthropic   # Priority 1 (primary)
  - name: gemini      # Priority 2 (fallback)

# If Anthropic fails → auto-retry on Gemini
# Health auto-recovers after 30 seconds
